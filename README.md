## 🚀 Featured Projects

### [Pulsefolio](https://pulsefolio.app/) — Next.js, React, TypeScript, Supabase, Tailwind CSS, Python

Pulsefolio is a portfolio-aware finance application that helps users connect market news with their personal investments.

The app ingests market and financial news, enriches articles with AI, matches relevant stories to user portfolios and watchlists, and generates personalized portfolio insights.

**Main features:**

* Portfolio import and holdings management
* Personalized market news feed
* AI-powered article enrichment
* Portfolio-specific matching and scoring
* Article-level chat
* Portfolio copilot chat
* Supabase authentication and database
* Python-powered news ingestion workflow

### Earnit! — Android, Java / Kotlin

Earnit! is a fitness productivity Android app designed to help users stay disciplined during workout rest periods.

The app works as a smart rest timer that helps users avoid distractions and return to training when the rest period is over.

**Main features:**

* Workout rest timer
* Focus-based decision screen
* Accessibility-based app redirection
* Local-first MVP
* No account required
* No ads
* No payments
* Built for discipline, fitness, and productivity
